# CMKDOWN - MP4/MP3 Downloader

Download MP4 and MP3 files from YouTube, Facebook, and Instagram.

## Features
- Download videos from YouTube as MP4
- Download audio from YouTube as MP3
- Download videos from Facebook
- Download videos from Instagram
- Background downloads
- Download history
- File management

## Requirements
- Android 8.0 (API level 26) or higher
- Internet permission
- Storage permission

## Installation
Clone this repository and build using Android Studio.

```bash
git clone https://github.com/cmkartproject/Mp4Mp3.git
cd Mp4Mp3
```

Open in Android Studio and run on an emulator or device.
