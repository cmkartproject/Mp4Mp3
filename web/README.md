# CMKDOWN Web Version

This is the web version of CMKDOWN - a video and audio downloader for YouTube, Facebook, and Instagram.

## Features

- 🎬 Download videos from YouTube
- 🎵 Extract audio (MP3) from YouTube videos
- 📱 Download videos from Facebook
- 📸 Download videos from Instagram
- 💾 Download history tracking
- 📊 Progress tracking
- 🎨 Modern, responsive UI

## How to Use

1. **Open index.html** in your web browser
2. **Paste a video URL** from:
   - YouTube (youtube.com or youtu.be)
   - Facebook (facebook.com)
   - Instagram (instagram.com)
3. **Select format**:
   - MP4 for video
   - MP3 for audio (YouTube only)
4. **Click Download**
5. **View download history** in the History tab

## Files

- `index.html` - Main HTML structure
- `css/style.css` - Styling and responsive design
- `js/app.js` - Application logic and interactivity

## Browser Support

- Chrome/Edge (recommended)
- Firefox
- Safari
- Mobile browsers

## Note

This is a demo/preview version. The actual download happens in the Android app which uses proper download APIs. The web version provides:
- URL validation
- Format selection interface
- Download progress simulation
- History tracking

For actual downloading functionality, use the Android app or the Termux build.
